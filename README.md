# Feiertags_Webseite

Moderne, responsive Webseite zur Anzeige von Feiertagen für:
- Nordrhein-Westfalen (NRW)
- Niedersachsen

## Features
- Auswahl des Bundeslands
- Liste aller Feiertage des aktuellen Jahres
- Countdown in Tagen pro Feiertag
- Nächster Feiertag prominent hervorgehoben
- Jahreswechsel-Logik für den nächsten Feiertag (wenn dieses Jahr vorbei, wird nächstes Jahr genutzt)

## Start
Einfach `index.html` im Browser öffnen.

Optional per lokalem Server:

```bash
cd Feiertags_Webseite
python3 -m http.server 8080
```

Dann öffnen: http://localhost:8080
